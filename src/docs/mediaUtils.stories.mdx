# Media utils


All the infrastructure, related to the OptimizedImage are located in the @interdan/media-utils package, which is being reexposed by @interdan/admin-ui.


Media utils documentation is provided [here](https://media-utils.intb.dk/?path=/story/installation-guide--page).


No additional setup steps necessary, just import OptimizedImage directly from @interdan/admin-ui.


``` ts

import { OptimizedImage } from '@interdan/admin-ui';

```
