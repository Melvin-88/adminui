# Color Palette

In our admin projects we try to stick to a basic set of colors  
So, this library exposes them all as `css variables`:

<ColorPalette>
  {Object.keys(colorPalette).map(color => (
    <ColorItem
      title={color}
      colors={colorPalette[color]}
    />
  ))}
</ColorPalette>
